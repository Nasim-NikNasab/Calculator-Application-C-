# Calculator Application

## About the Project

This project is a **Calculator Application** developed with **C#**, **.NET Framework**, and **Windows Forms**. It provides a graphical user interface with two calculator modes: **Standard** and **Scientific**.

The application is designed to perform common arithmetic operations as well as several advanced mathematical functions through an easy-to-use interface.

## Features

### Standard Calculator

* Addition
* Subtraction
* Multiplication
* Division
* Modulus operation
* Square root (√)
* Square (x²)
* Decimal number support
* Positive/Negative number conversion
* Backspace
* Clear input

### Scientific Calculator

* Power (xʸ)
* Factorial (n!)
* Reciprocal (1/x)
* Natural logarithm (ln)
* Logarithm (log)
* Sine (sin)
* Cosine (cos)

### User Interface

* Graphical interface built with Windows Forms
* Switch between Standard and Scientific calculator modes
* Input validation
* Error messages for invalid operations such as division by zero and negative square roots

## Technologies Used

* C#
* .NET Framework
* Windows Forms
* Visual Studio

## Project Structure

* **Form1** – Standard Calculator
* **Form2** – Scientific Calculator
* **Program.cs** – Application entry point

## How to Run

1. Open the solution in Visual Studio.
2. Build the project.
3. Run the application.
4. Use the Standard or Scientific calculator to perform calculations.
